# Security Policy

## Reporting a vulnerability

If you believe you found a security issue in Outreach, please report it privately.

- Preferred contact: **REPLACE_WITH_SECURITY_EMAIL** (set this to your real security contact)
- Do not open a public GitHub issue for sensitive reports

Please include:

1. A short description of the issue
2. Steps to reproduce
3. Impact assessment (what an attacker could do)
4. Any suggested fix, if you have one

## Responsible disclosure

We ask that you give us a reasonable time to investigate and fix the issue before any public disclosure. We will acknowledge receipt and keep you updated as we work through a fix.
